# NyLittleScrunchies

Project structure:

- `index.html` - homepage
- `products.html` - product listing
- `product-detail.html` - product details
- `panier.html` - cart and checkout
- `confirmation.html` - order confirmation
- `assets/css/style.css` - global styles
- `assets/js/components.js` - shared UI components
- `assets/js/data.js` - product catalog data
- `assets/js/script.js` - app logic
- `assets/images/` - product images

Notes:

- Images are stored in `assets/images/` to keep media separated from source files.
- CSS and JavaScript now live under `assets/` for a cleaner production-ready layout.
